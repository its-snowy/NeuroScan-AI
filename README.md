## 🔬 Overview

NeuroScan-AI is a groundbreaking Flutter application that harnesses the power of artificial intelligence to analyze brain MRI scans with unparalleled accuracy. By combining advanced machine learning techniques with a user-friendly mobile interface, we've created a powerful tool for brain MRI classification and analysis.

## 🌟 Key Features

🎯 **High Accuracy**: 96% accuracy in classifying brain MRI scans
📱 **User-Friendly Interface**: Intuitive Flutter-based Android app
🤖 **NeuroScan Bot**: AI-powered chatbot for personalized symptom insights
📊 **Comprehensive Reporting**: Detailed analysis with easy-to-understand visualizations
🔒 **Data Security**: End-to-end encryption for patient data protection
📸 **Multi-Input Support**: Upload file or capture image (beta) for analysis

## 🧠 Machine Learning Model

Our state-of-the-art machine learning model is the core of NeuroScan-AI, providing highly accurate brain MRI scan classifications:

- **Architecture**: Utilizes the VGG16 convolutional neural network architecture
- **Transfer Learning**: Leverages TensorFlow's transfer learning capabilities for enhanced performance
- **Training Data**: Trained on a diverse dataset of brain MRI scans to ensure robust classification
- **Accuracy**: Achieves an impressive 96% accuracy rate in classifying brain MRI scans
- **Input Flexibility**: Accepts both uploaded files and captured images (beta feature) for analysis
- **Continuous Improvement**: Regular model updates based on new data and emerging research

## 📱 Flutter Android App

The NeuroScan-AI Flutter Android app provides a seamless user experience for interacting with our powerful ML model:

- **Intuitive UI**: Clean, user-friendly interface designed for healthcare professionals
- **Secure Authentication**: Robust user authentication and authorization system
- **Scan Upload**: Easy-to-use interface for uploading MRI scan files or capturing images
- **Real-time Processing**: Instantaneous processing and classification of uploaded scans
- **Result Visualization**: Clear and concise presentation of classification results
- **History Tracking**: Maintain a record of previous scans and classifications
- **Cross-platform Potential**: Built with Flutter for future iOS and web expansion

## 🤖 NeuroScan Bot

The NeuroScan Bot is an AI-powered chatbot that complements the MRI classification results:

- **Symptom Analysis**: Provides insights on potential symptoms based on classification results
- **Natural Language Processing**: Understands and responds to user queries in natural language
- **Personalized Interaction**: Tailors responses based on user's scan results and medical history
- **Information Source**: Offers evidence-based information from reputable medical sources
- **Continuous Learning**: Improves responses over time through machine learning algorithms
- **Multilingual Support**: Communicates in multiple languages for global accessibility

## 🚀 Getting Started

Get NeuroScan-AI up and running on your local machine in just a few steps:

```bash
# Clone the repository
git clone https://github.com/its-snowy/NeuroScan-AI.git

# Navigate to the project directory
cd NeuroScan-AI

# Install dependencies
flutter pub get

# Run the application
flutter run
```

## 💻 Tech Stack

- **Flutter**: UI toolkit for building natively compiled applications
- **Dart**: Programming language for Flutter app development
- **TensorFlow**: Open-source machine learning framework for model development
- **Python**: Used for data preprocessing and model training
- **Firebase**: Backend services for authentication and data storage
- **DialogFlow**: Natural language processing for NeuroScan Bot
